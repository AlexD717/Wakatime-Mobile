# Wakatime Mobile

This is a simple mobile app for android devices, allowing you to easily view your WakaTime stats

## Setup

Install the latest .apk from the releases tab
When first launching the app it will ask you for a api key, this can be found here: [https://wakatime.com/settings/account](https://wakatime.com/settings/account) _(Note: Your key should start with waka_)_

_Please note, that this app isn't compatible with Hackatime, only WakaTime_

## Features
- Real time statistics about coding time and top languages
- Language breakdown with pie charts
- List of projects with amount of time spent on each
- Customizable time ranges
